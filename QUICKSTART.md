# 快速开始 - 5 分钟上手 Tasksboard

## 🚀 一键安装

### 1. 打开扩展页面
在 Chrome 地址栏输入：`chrome://extensions/` 然后回车

### 2. 加载插件
1. 打开右上角「开发者模式」
2. 点击「加载已解压的扩展程序」
3. 选择本项目文件夹
4. 完成！

### 3. 打开侧边栏
点击工具栏的 Tasksboard 图标（蓝色勾选图标）

---

## 📝 基本使用（30 秒学会）

### 添加任务
点击 **「+ Add a task」** → 输入内容 → 按 Enter

### 编辑任务
点击任务文本 → 修改内容 → 按 Enter 或点击外部

### 设置优先级
点击任务左侧的优先级标记（—），循环切换：
- 🔴 **S** - 超重要
- 🟠 **A** - 重要
- 🔵 **B** - 普通
- ⚪ **C** - 低优先级
- **—** - 无优先级

### 调整顺序
拖拽任务左侧的 **⋮⋮** 图标到目标位置

### 完成任务
点击任务前的 **○** 复选框，任务移到底部「Completed」区域

### 删除任务
悬停任务 → 点击右侧的 **×** 按钮

---

## 💡 使用技巧

### 快速创建多个任务
1. 点击「+ Add a task」
2. 输入第一个任务
3. 按 Enter 保存
4. 再次点击「+」添加下一个

### 批量设置优先级
创建所有任务后，从上到下点击优先级标记快速设置

### 保持界面整洁
完成任务后自动折叠「Completed」区域，点击标题可展开查看

### 重要任务置顶
拖拽高优先级任务到列表顶部，一目了然

## 🔧 遇到问题？

### 侧边栏打不开？
→ 确保 Chrome 版本 ≥ 114，更新到最新版本

### 点击图标没反应？
→ 在 `chrome://extensions/` 点击「重新加载」刷新插件

### 任务没有保存？
→ 打开 DevTools (F12) 查看 Console 是否有错误

### 想清除所有数据？
→ DevTools → Application → Local Storage → 删除数据
