# UI 设计指南 - Tasksboard 界面说明

## 📐 界面布局

```
┌─────────────────────────────────────────┐
│  Tasksboard                             │  ← 标题
│  今日任务                                │  ← 副标题
├─────────────────────────────────────────┤
│  [+ Add a task]                         │  ← 快速添加按钮
├─────────────────────────────────────────┤
│                                         │
│  ⋮⋮ ○ [S] 工作：完成项目报告           │  ← 任务 1
│  ⋮⋮ ○ [A] 学习：阅读技术文档           │  ← 任务 2
│  ⋮⋮ ○ [B] 生活：购买日用品             │  ← 任务 3
│  ⋮⋮ ○ [—] 其他：打扫房间               │  ← 任务 4
│                                         │
│  (可滚动区域)                            │
│                                         │
├─────────────────────────────────────────┤
│  Completed (2) ▼                        │  ← 已完成标题（可折叠）
│  ⋮⋮ ✓ 购买食材                         │  ← 已完成任务
│  ⋮⋮ ✓ 回复邮件                         │
└─────────────────────────────────────────┘
```

---

## 🎨 颜色方案

### 主色调
```css
背景色：    #FFFFFF (白色)
文本色：    #202124 (深灰黑)

边框色：    #DADCE0 (浅灰)
悬停色：    #F1F3F4 (极浅灰)
激活色：    #E8F0FE (淡蓝)
```

### 优先级颜色
```css
S (超重要)： #EA4335 (Google Red)    🔴
A (重要)：   #F9AB00 (Google Orange) 🟠
B (普通)：   #4285F4 (Google Blue)   🔵
C (低优先级)：#80868B (灰色)         ⚪
无优先级：   虚线边框 + —
```

## 🧩 组件详解

### 1. 页头 (Header)

```
┌─────────────────────┐
│  Tasksboard         │  ← 24px, 粗体 #202124
│  今日任务            │  ← 13px, 常规 #5F6368
└─────────────────────┘
```

**样式特征：**
- 固定在顶部
- 底部留白 16px
- 简洁无边框

---

### 2. 添加按钮

```
┌─────────────────────────────┐
│  + Add a task               │  ← 14px, #5F6368
└─────────────────────────────┘
```

**交互：**
- 默认：白色背景 + 灰色边框
- 悬停：浅灰背景 + 深灰边框
- 点击：创建新任务

**尺寸：**
- 宽度：100%
- 内边距：12px
- 圆角：8px

---

### 3. 任务项 (Task Item)

```
┌──────────────────────────────────────┐
│ ⋮⋮  ○  [S]  工作：完成项目报告  ×   │
│ │   │   │    └─ 内容            │   │
│ │   │   └── 优先级              └─ 删除
│ │   └────── 复选框
│ └────────── 拖拽把手
└──────────────────────────────────────┘
```

#### A. 拖拽把手 (⋮⋮)
- **尺寸：** 16px
- **颜色：** #5F6368
- **功能：** 鼠标按住拖拽排序
- **状态：** 已完成任务隐藏

#### B. 复选框 (○/✓)
- **尺寸：** 18x18px
- **默认：** 白色 + 灰色边框（圆形）
- **悬停：** 蓝色边框
- **选中：** 蓝色填充 + 白色勾选 ✓
- **功能：** 切换完成状态

#### C. 优先级徽章
```
┌────┐  ┌────┐  ┌────┐  ┌────┐  ┌─ ─ ┐
│ S  │  │ A  │  │ B  │  │ C  │  │  —  │
└────┘  └────┘  └────┘  └────┘  └─ ─ ┘
红色    橙色    蓝色    灰色    虚线
```

- **尺寸：** 24x24px
- **圆角：** 4px
- **字体：** 11px 粗体
- **交互：** 点击循环切换
- **悬停：** 放大 1.1 倍

#### D. 任务内容
- **字体：** 14px 常规
- **颜色：** #202124（未完成）/ #5F6368（已完成）
- **装饰：** 已完成显示删除线
- **编辑：** 点击进入编辑状态
- **编辑时：** 蓝色边框 + 浅蓝背景

#### E. 删除按钮 (×)
- **尺寸：** 18px
- **颜色：** #5F6368（默认）/ #EA4335（悬停）
- **显示：** 仅悬停任务时显示
- **位置：** 右侧对齐

---

### 4. 已完成区域

```
┌─────────────────────────────┐
│  Completed (2) ▼            │  ← 可折叠标题
├─────────────────────────────┤
│  ⋮⋮ ✓ 任务 1                │  ← 已完成任务
│  ⋮⋮ ✓ 任务 2                │
└─────────────────────────────┘
```

#### 标题栏
- **字体：** 13px 中粗
- **颜色：** #5F6368
- **图标：** ▼（展开）/ ▶（折叠）
- **悬停：** 浅灰背景
- **功能：** 点击折叠/展开

#### 已完成任务
- **样式：** 灰色 + 删除线
- **拖拽：** 禁用
- **编辑：** 禁用
- **删除：** 允许

---

### 5. 空状态

```
┌─────────────────────────────┐
│                             │
│         暂无任务             │  ← 14px #5F6368
│    点击上方添加第一个任务     │  ← 12px #5F6368
│                             │
└─────────────────────────────┘
```

**显示条件：**
- 无活动任务 && 无已完成任务

## 🎬 动画效果

### 1. 淡入动画 (fadeIn)
```css
from: opacity 0, translateY(-10px)
to:   opacity 1, translateY(0)
duration: 250ms
```
**应用场景：** 创建新任务

### 2. 淡出动画 (fadeOut)
```css
from: opacity 1, translateY(0)
to:   opacity 0, translateY(-10px)
duration: 250ms
```
**应用场景：** 删除任务

### 3. 拖拽效果
```css
dragging: opacity 0.5
```
**应用场景：** 拖拽任务时

### 4. 折叠动画
```css
transition: max-height 250ms ease
collapsed: max-height 0
expanded: max-height 300px
```
**应用场景：** 折叠/展开已完成区域

### 5. 优先级悬停
```css
hover: transform scale(1.1)
transition: 150ms ease
```
**应用场景：** 鼠标悬停优先级徽章

## 🖱️ 交互状态

### 悬停 (Hover)
| 元素 | 效果 |
|------|------|
| 任务项 | 背景 → #F1F3F4 |
| 按钮 | 背景 → #F1F3F4 + 深色边框 |
| 优先级 | 放大 1.1 倍 |
| 删除按钮 | 红色 #EA4335 |

### 激活 (Active/Focus)
| 元素 | 效果 |
|------|------|
| 编辑中任务 | 蓝色边框 + 浅蓝背景 #E8F0FE |
| 拖拽中任务 | 半透明 opacity 0.5 |

### 禁用 (Disabled)
| 元素 | 效果 |
|------|------|
| 已完成任务 | 不可拖拽、不可编辑 |
| 已完成优先级 | 不可点击 |

## 📱 响应式设计

### 侧边栏宽度
```css
最小宽度：  320px
最大宽度：  400px
推荐宽度：  ~350px
```

### 滚动区域
- **任务列表：** 自动滚动，最大高度 = 视口高度 - 头部 - 底部
- **已完成列表：** 固定最大高度 300px

### 滚动条样式
```css
宽度：    8px
背景：    透明
滑块：    #DADCE0 (默认) / #5F6368 (悬停)
圆角：    4px
```

## 🎯 设计原则

### 1. 简洁至上
- 无冗余元素
- 扁平化设计
- 留白充足

### 2. 操作直观
- 所有功能点击即可完成
- 无需学习成本
- 视觉提示清晰

### 3. 视觉层次
- 优先级用颜色区分
- 已完成任务用灰色区分
- 重要操作（添加、删除）位置固定

### 4. 性能优先
- CSS 动画优于 JS 动画
- 避免过度动画
- 流畅的拖拽体验

---

## 🔍 细节打磨

### 1. 无障碍支持
- 充足的点击区域（最小 24x24px）
- 清晰的颜色对比度
- 键盘导航支持（Enter, Escape）

### 2. 边界处理
- 超长文本自动换行
- 空状态友好提示
- 错误信息清晰

### 3. 视觉反馈
- 所有交互都有反馈
- 悬停、点击、拖拽均有视觉变化
- 状态切换平滑过渡
