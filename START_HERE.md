## ⚡ 5 分钟快速开始

### 第一步：安装插件

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 打开「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹
6. 完成！

### 第二步：使用插件

1. 点击工具栏的 Tasksboard 图标
2. 点击「+ Add a task」创建任务
3. 点击任务内容编辑
4. 点击优先级标记设置重要程度
5. 拖拽任务调整顺序

### 第三步：享受高效！

现在你已经掌握了所有核心功能！🎉

## ❓ 常见问题

### Q1: 安装后找不到图标？
**A:** 点击浏览器工具栏的拼图图标 → 找到 Tasksboard → 固定到工具栏

### Q2: 侧边栏打不开？
**A:** 确保 Chrome 版本 ≥ 114，更新到最新版本

### Q3: 数据会丢失吗？
**A:** 所有数据存储在本地，除非卸载插件或清除浏览器数据，否则不会丢失

### Q4: 如何备份数据？
**A:** 未来版本将支持导出功能，当前可通过 DevTools 查看 Local Storage 手动备份

### Q5: 可以在其他浏览器使用吗？
**A:** 目前仅支持 Chrome/Edge（Chromium 内核），Firefox 需要适配

### Q6: 如何修改代码？
**A:** 参考 `CLAUDE.md` 开发文档，修改后在 `chrome://extensions/` 重新加载

### Q7: 发现 Bug 怎么办？
**A:** 打开 DevTools 查看 Console 错误，参考 `TESTING_CHECKLIST.md` 验证
